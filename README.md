# Object-Oriented Menu and Billing System
A C++ command-line application developed for a couse at Seneca Polytechnic, focusing on modular design and dynamic memory management.

## Technical Highlights
* OOP Principles: Implemented Encapsulation and Modular Design.
* Memory Management: Handled dynamic memory allocation for menu items and order tracking.
* File I/O: Loads restaurant data from `.csv` files and generates formatted bills.
